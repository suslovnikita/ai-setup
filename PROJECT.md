# Time Booking Service

## Описание продукта

Веб-приложение для бронирования временных слотов. Пользователи выбирают доступное время и записываются на услуги (консультации, встречи и т.д.), а владельцы управляют расписанием и доступностью.

---

## MVP (Minimum Viable Product)

* Регистрация и авторизация
* Просмотр доступных слотов
* Бронирование слота
* Отмена бронирования
* Создание слотов (админ)

---

## Core Features

* Управление расписанием (создание, редактирование, удаление слотов)
* Ограничение длительности слотов
* Фильтрация по дате
* История бронирований
* Ограничение: 1 слот на пользователя/владельца в день

## Use Cases

### Пользователь

* Просматривает доступные слоты
* Выбирает удобное время
* Бронирует слот
* Отменяет бронирование
* Просматривает историю записей

### Владелец (админ)

* Создает временные слоты
* Управляет расписанием
* Просматривает список бронирований
* Контролирует доступность времени
